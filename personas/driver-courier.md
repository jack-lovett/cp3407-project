# Persona: Driver / Courier

- **Role / Type:** Driver
- **Primary goal:** Complete deliveries efficiently with clear pickup/drop-off instructions.
- **Secondary goals:** Minimise wait times; avoid wrong addresses; maintain rating.
- **Context:** Mobile app, on the move.
- **Behaviours:** Accepts jobs, navigates, confirms pickup/drop-off, updates status.
- **Pain points:** Restaurant delays; unclear delivery notes; address issues.
- **Motivations:** Efficiency, predictable workflow, fewer disputes.
- **Tech comfort:** medium

## Core jobs-to-be-done
- When I’m doing deliveries, I want clear tasks and statuses, so I can pick up and drop off orders without confusion.

## Key user journeys (high-level)
- Receive assignment → navigate to pickup → confirm pickup → deliver → confirm drop-off.

## Top needs / priorities
- P20: Clear status updates for customers (driver-triggered).
- P30: Contact support easily.
- P40: Clear delivery details.
